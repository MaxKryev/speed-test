# Speed Test

Консольный скрипт для замера скорости интернета на Python. Выполняет последовательные HTTP-запросы к указанному URL, измеряет время загрузки и вычисляет среднюю скорость скачивания.

## Особенности

### Функциональность
- **Последовательные запросы**: Выполняет 10 (по умолчанию) запросов к указанному URL для получения статистически значимых результатов.
- **Точное измерение**: Использование `time.perf_counter()` для высокоточного замера времени.
- **Потоковая загрузка**: Чтение данных чанками через `stream=True` для корректного измерения объема.
- **Обработка ошибок**: Graceful handling сетевых ошибок с продолжением теста.

### Качество кода
- **Типизация**: Полное покрытие аннотациями типов (typing module).
- **Чистая архитектура**: Разделение логики на функции (измерение, агрегация, вывод).
- **CLI интерфейс**: argparse для удобного управления параметрами из командной строки.

### Отчетность
- **Детальная статистика**: Успешные запросы, общий объем, суммарное время.
- **Средние значения**: Среднее время запроса, средний размер файла.
- **Скорость**: Расчет скорости в MB/s (мегабайты в секунду).

## Установка

### 1. Клонировать репозиторий
```bash
git clone https://github.com/username/speed-test.git
cd speed-test
```

### 2. Установить зависимости через Poetry
```bash
poetry install
```

### 3. Активировать виртуальное окружение (опционально)
```bash
poetry shell
```

## Запуск

### Базовый запуск (URL по умолчанию)
```bash
poetry run python app/main.py
```

### С указанием URL
```bash
poetry run python app/main.py https://example.com/large-file.jpg
```

### С настройкой количества запросов
```bash
poetry run python app/main.py -n 5
poetry run python app/main.py https://example.com/file.zip -n 20
```

## Примеры использования
1. Тест с легкой картинкой (по умолчанию)
```bash
poetry run python app/main.py
```
2. Тест с тяжелым GIF

Заменить в модуле main.py DEFAULT_URL на HEAVY_URL (строка 75) и выполнить команду
```bash
poetry run python app/main.py
```
